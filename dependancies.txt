development environment

get Lubuntu 64 bit ISO
make sure Virtualization Technique or VT is enabled in system bios
install virtualbox
install lubuntu
boot it up
sudo apt-get install build-essential
mount the vbox-guest-addition iso from the virtualbox menu
sudo apt-get install virtualenv
cd to a folder where you want a virtualenv
virtualenv --system-site-packages python-2.7-system
cd to the bin folder and: source activate
pip install -U pip #to fix pip freeze issue
sudo apt-get install python-dev
sudo apt-get install -y mongodb #-y means yes to all
in virtualbox go to VMs settings and port forward host 8888 guest 8888
now things like web2py can be run in lubuntu guest, and accessed at host guest via 127.0.0.1, 192.168..., or 0.0.0.0

add a folder:
go to VMs settings and add folder, enter folder path, and give it a name
set it to make permanent and automount
then in lubuntu guest, type:
sudo adduser <login name here> vboxsf
reboot
now folder should be in /mnt
